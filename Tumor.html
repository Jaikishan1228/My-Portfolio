Brain Tumor Detection — YOLOv9

Custom-trained YOLOv9 object detection model to detect & localize brain tumors in medical MRI/scan images. Workflow includes dataset preparation in YOLO format, training, evaluation visualizations and an inference notebook.

View on GitHub Back to Projects
Quick Facts
  • Primary framework: ultralytics YOLOv9 (custom training)
  • Notebook: YOLOv9_BrainTumor_Detection.ipynb

How to Run (quick)

  1. Prepare dataset in YOLO format (images + .txt labels). See notebook for expected folder structure.
  2. Install requirements: Python 3.8+, ultralytics, opencv-python, matplotlib.
  3. Open and run YOLOv9_BrainTumor_Detection.ipynb — it contains dataset prep, training and inference cells.
  4. Trained weights are saved to models/yolov9_brain_tumor.pt. Inference outputs are saved under runs/cancer_detection*.
