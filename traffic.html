Project Overview

This project demonstrates forecasting website traffic (daily visitors, pageviews, sessions) by analyzing historical traffic logs, identifying trends and seasonality, and comparing classical time-series and machine-learning approaches. It aims to produce reliable short-to-medium term forecasts useful for planning marketing campaigns, capacity planning, and business insights.

Features

  • Data preprocessing and cleaning of website traffic logs
  • Exploratory Data Analysis (trend, seasonality, decomposition)
  • Time-series forecasting using ARIMA / SARIMA and Prophet
  • Machine learning regression model baselines
  • Model evaluation with RMSE, MAPE and MAE; comparison plots of predicted vs. actual

How to run (quick)

  1. Clone the repository: 
    git clone https://github.com/Jaikishan1228/Forecasting_website_traffic_SI_05.git
  2. Install dependencies: 
    pip install -r requirements.txt
  3. Open the notebook: 
    jupyter notebook Forecasting_website_traffic.ipynb
  4. Run the notebook cells to preprocess data, fit models, and generate forecast plots.
