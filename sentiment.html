Project Overview

This project analyzes user sentiment in social media posts to determine whether the opinions expressed are positive, negative, or neutral. It applies NLP preprocessing (tokenization, stopword removal, stemming/lemmatization), classical ML classifiers and visualization to surface trends and frequent terms.

Features

  • Text preprocessing pipeline (tokenize, stopword removal, stemming/lemmatization)
  • Feature extraction (TF-IDF / Bag-of-Words)
  • Sentiment classification with machine learning models (scikit-learn)
  • Evaluation (accuracy, precision, recall, F1-score)
  • Visualizations: sentiment distribution, trend plots, word clouds

How to run (quick)

  1. Clone the repo: git clone https://github.com/Jaikishan1228/Social_media_sentiment_analysis_SI_03.git
  2. Install dependencies: pip install -r requirements.txt
  3. Open the notebook: jupyter notebook Social_Media_Sentiment_Analysis.ipynb
  4. Run cells for data loading, preprocessing, training and visualization.
Requirements
Python 3, pandas, numpy, nltk/spacy, scikit-learn, matplotlib, seaborn, jupyter

Evaluation

The notebook produces model performance outputs (accuracy, precision, recall, F1-score). See the notebook for full evaluation tables and confusion matrices.
