RC Plane Flight Controller (Arduino Nano + ADXL345)

Build Summary & Wiring

Overview of wiring and key connections used in this build. This is a compact flight controller based on an Arduino Nano reading ADXL345 accelerometer data, using PID stabilization to drive aileron/elevator servos. The receiver channels map controls and safety failsafe features are implemented in firmware.

Wiring (high level)
  • Battery → ESC → Motor: Connect LiPo to ESC battery leads; motor 3-phase wires to ESC output.
  • ESC signal: ESC servo signal wire → throttle channel on receiver (or to Arduino PWM output if passing throttle through microcontroller).
  • Receiver → Arduino: Map receiver PWM (or SBUS) channels to Arduino inputs. Typical mapping: throttle, aileron, elevator, rudder (if used), flight mode.
  • Arduino ↔ ADXL345: Connect ADXL345 to Arduino via I2C (SDA, SCL), 3.3V (or 5V depending on module), GND. Use I2C addresses and wiring per module docs.
  • Arduino → Servos: Arduino PWM outputs to aileron & elevator servos (ensure common ground with receiver and battery). Use a separate BEC or ESC-provided 5V regulator for servo power if needed.
Suggested pin mapping (example)
Arduino Nano (ATmega328P)
  A4 (SDA)  → ADXL345 SDA
  A5 (SCL)  → ADXL345 SCL
  D3        → Aileron servo PWM
  D5        → Elevator servo PWM
  D6        → (Optional) LED / status
  GND       → Common ground (battery negative, ESC BEC, receiver)
  5V / 3.3V → Power to ADXL345 (match module voltage)
Notes
  • Use a common ground between the Arduino, receiver, ESC (BEC) and servos.
  • If your ESC does not provide BEC / 5V for servos, use a separate 5V regulator / BEC rated for servo current.
  • Check receiver protocol — if using SBUS/iBUS, you may need a UART serial parser or use a flight receiver with PWM outputs.

Firmware, Control & Tuning

Firmware runs on the Arduino Nano and implements a basic attitude-stabilization loop:

  1. Read accelerometer (ADXL345) and receiver inputs (PWM or serial).
  2. Estimate roll/pitch from accelerometer; optionally add complementary filter if using gyro later.
  3. Compute PID control for aileron/elevator based on desired stick inputs vs measured attitude.
  4. Output PWM to servos for control surfaces. Throttle usually passed through to ESC (unless motor speed is controlled directly).
Calibration & Tuning
  • Place plane on level surface and record raw ADXL345 readings to set offsets (zero-g calibration).
  • Tune PID gains on the ground with props removed (very small gains to verify direction of control).
  • Do low-altitude test flights in manual mode; gradually increase stabilization gains until stable response is achieved.
  • Record flight logs (serial prints) for later analysis and iterative tuning.
